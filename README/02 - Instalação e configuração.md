# 02 - Instalação e configuração

## Instalação

1 - Colar a pasta dist dentro do drive C: ou onde desejar, e renomea-la para FolSyn

2 - Criar um atalho do arquivo **FolSyn_v1.exe**

3 - Colocar o atalho em Publico

## Configuração

Na pasta junto do arquivo **sat_v1_7.exe** é importante ter dois itens:

- pasta ***database***: pasta que vai receber o arquivo de banco de dados.
- arquivo **source_folder.txt**: arquivo de texto comum, com o caminho local da pasta que deve ser sincronizada. Ou seja, a pasta que vai ser replicada para todas as pastas cadastradas na lista.

    Exemplo:  **E:\Audios ou D:\Fotos**

    Importante: nesse arquivo só deve estar o caminho para uma única pasta.

    ![02%20-%20Instalac%CC%A7a%CC%83o%20e%20configurac%CC%A7a%CC%83o/Untitled.png](02%20-%20Instalac%CC%A7a%CC%83o%20e%20configurac%CC%A7a%CC%83o/Untitled.png)

- Para que a sincronização com uma pasta tenha sucesso, é preciso que a pasta destino esteja compartilhada na rede.
